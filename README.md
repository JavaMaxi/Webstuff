<h1 id="user-content-overview">Overview</h1>
<p>Red is a fully modular bot – meaning all features and commands can be enabled/disabled to your
liking, making it completely customizable. This is also a <em>self-hosted bot</em> – meaning you will need
to host and maintain your own instance. You can turn Red into an admin bot, music bot, trivia bot,
new best friend or all of these together!</p>
<p><a href="#user-content-installation" rel="nofollow">Installation</a> is easy, and you do <strong>NOT</strong> need to know anything about coding! Aside
from installation and updating, every part of the bot can be controlled from within Discord.</p>
<p><strong>The default set of modules includes and is not limited to:</strong></p>
<ul>
<li>Moderation features (kick/ban/softban/hackban, mod-log, filter, chat cleanup)</li>
<li>Trivia (lists are included and can be easily added)</li>
<li>Music features (YouTube, SoundCloud, local files, playlists, queues)</li>
<li>Stream alerts (Twitch, Youtube, Mixer, Hitbox, Picarto)</li>
<li>Bank (slot machine, user credits)</li>
<li>Custom commands</li>
<li>Imgur/gif search</li>
<li>Admin automation (self-role assignment, cross-server announcements, mod-mail reports)</li>
<li>Customisable command permissions</li>
</ul>
<p><strong>Additionally, other <a href="#user-content-plugins" rel="nofollow">plugins</a> (cogs) can be easily found and added from our growing
community of cog repositories.</strong></p>
<h1 id="user-content-installation">Installation</h1>
<p><strong>The following platforms are officially supported:</strong></p>
<ul>
<li><a href="https://red-discordbot.readthedocs.io/en/stable/install_windows.html" rel="nofollow">Windows</a></li>
<li><a href="https://red-discordbot.readthedocs.io/en/stable/install_linux_mac.html" rel="nofollow">MacOS</a></li>
<li><a href="https://red-discordbot.readthedocs.io/en/stable/install_linux_mac.html" rel="nofollow">Most major linux distributions</a></li>
</ul>
<p>If after reading the guide you are still experiencing issues, feel free to join the
<a href="https://discord.gg/red" rel="nofollow">Official Discord Server</a> and ask in the <strong>#support</strong> channel for help.</p>
<h1 id="user-content-plugins">Plugins</h1>
<p>Red is fully modular, allowing you to load and unload plugins of your choice, and install 3rd party
plugins directly from Discord! A few examples are:</p>
<ul>
<li>Cleverbot integration (talk to Red and she talks back)</li>
<li>Ban sync</li>
<li>Welcome messages</li>
<li>Casino</li>
<li>Reaction roles</li>
<li>Slow Mode</li>
<li>AniList</li>
<li>And much, much more!</li>
</ul>
<p>Feel free to take a <a href="https://cogboard.red/t/approved-repositories/210" rel="nofollow">peek</a> at a list of
available 3rd party cogs!</p>
<h1 id="user-content-join-the-community">Join the community!</h1>
<p><strong>Red</strong> is in continuous development, and it’s supported by an active community which produces new
content (cogs/plugins) for everyone to enjoy. New features are constantly added. If you can’t
<a href="https://cogboard.red/t/approved-repositories/210" rel="nofollow">find</a> the cog you’re looking for,
consult our <a href="https://red-discordbot.readthedocs.io/en/stable/guide_cog_creation.html" rel="nofollow">guide</a> on
building your own cogs!</p>
<p>Join us on our <a href="https://discord.gg/red" rel="nofollow">Official Discord Server</a>!</p>
